This is a beginner-level automation project built using Java and Selenium WebDriver.
The application launches a browser, navigates to a website, reads the page title, and prints it in the terminal.

🚀 Project Objective

The main goal of this project is to understand the basics of:
Selenium WebDriver setup
Launching a browser using Java
Navigating to a webpage
Fetching the page title
Displaying output in the terminal

This project is ideal for students and beginners starting with automation testing.

🛠️ Technologies Used

🌐 Selenium WebDriver

🖥️ ChromeDriver

⚙️ Prerequisites

Before running this project, ensure you have:
Java (JDK 8 or above) installed
Selenium library added to your project
Chrome browser installed
ChromeDriver downloaded and configured

▶️ How to Run

Clone the repository:

git clone https://github.com/your-username/simple-selenium-project.git

Open the project in VS Code (or any Java IDE).

Make sure ChromeDriver path is correctly configured.

Run the main Java file.

🧾 Expected Output
The program will:
Open the browser
Navigate to the specified website
Print the page title in the terminal
Close the browser

🎯 Learning Outcomes
After completing this project, you will understand:
How Selenium interacts with browsers
Basic WebDriver methods
How to configure and run Selenium with Java
